# Slice 163: Canonical Widened-Suite Report

## Goal

Report the widened canonical manifest through explicit contexts for all stable
suite families.

## Shared Behavior

This slice defines one canonical widened-suite reporting contract:

1. the widened canonical manifest MAY be reported through explicit contexts for
   all stable suite families,
2. the canonical report envelope includes both config-family and source-family
   suite entries,
3. the widened canonical report continues to use one normalized manifest report
   shape.
