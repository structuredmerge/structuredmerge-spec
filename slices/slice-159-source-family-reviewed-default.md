# Slice 159: Source-Family Reviewed Default

## Goal

Apply an explicit review decision to the dedicated source-family review state.

## Shared Behavior

This slice defines one source-family reviewed-default contract:

1. a host MAY accept a synthesized default source-family context during review,
2. the accepted decision becomes an applied decision in the reviewed state,
3. the reviewed source-family report then includes the newly activated suite
   results while unresolved families remain review requests.
