## Goal

Apply an explicit review decision to the widened canonical review state.

This slice defines one canonical widened-suite reviewed-default contract:

1. a host MAY accept a synthesized default context for one widened canonical
   source family,
2. the accepted decision becomes an applied decision in the canonical reviewed
   state,
