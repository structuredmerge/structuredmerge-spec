# Slice 157: Canonical Stable-Suite Review State

## Goal

Review the stable config-family suites directly from the canonical manifest.

## Shared Behavior

This slice defines one canonical stable-suite review-state contract:

1. the canonical manifest MAY be reviewed with the stable config-family suites
   active,
2. defaultable family contexts MAY still surface review requests under strict
   review modes,
3. source-family suites remain absent from the reviewed canonical suite set.
