# Slice 202: Markdown Family Manifest Report

## Goal

Report a Markdown family manifest through host-specific planning contexts.

## Shared Behavior

This slice defines one Markdown manifest-reporting contract:

1. a Markdown family manifest MAY be reported through the existing manifest
   report helper,
2. host-specific Markdown contexts determine which suite plans are available,
3. the observable report remains the ordinary suite report envelope plus
   manifest diagnostics.
