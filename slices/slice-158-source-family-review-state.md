# Slice 158: Source-Family Review State

## Goal

Review the dedicated source-family manifest through the existing review-state
surface.

## Shared Behavior

This slice defines one source-family review-state contract:

1. the dedicated source-family manifest MAY be reviewed through the existing
   review helper,
2. explicit source-family contexts MAY coexist with missing contexts that become
   review requests,
3. missing source-family contexts remain explicit requests rather than silent
   omissions.
