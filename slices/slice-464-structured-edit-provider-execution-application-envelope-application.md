# Slice 464: Structured Edit Provider Execution Application Envelope Application

## Goal

Apply a supported provider execution-application envelope by importing it into
the shared application shape.

## Shared Behavior

This slice defines one shared application contract:

1. a supported envelope imports to the original provider execution
   application,
2. rejected envelopes preserve the slice-463 rejection behavior.

## Notes

- This slice standardizes envelope application, not provider dispatch.
