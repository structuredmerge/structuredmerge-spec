# Slice 201: Markdown Family Named-Suite Plans

## Goal

Plan Markdown named suites through host-specific family plan contexts.

## Shared Behavior

This slice defines one Markdown named-suite planning contract:

1. a Markdown named suite MAY be planned through the existing named-suite
   planning helpers,
2. the selected Markdown family plan context contributes the host-specific
   feature profile to each planned case run,
3. the suite identity and Markdown role order stay stable across hosts.
