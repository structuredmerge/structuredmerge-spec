# Slice 200: Markdown Family Suite Definitions

## Goal

Expose the Markdown family through the shared named-suite mechanism.

## Shared Behavior

This slice defines one Markdown suite-definition contract:

1. a Markdown manifest MAY define a named suite such as `markdown_portable`,
2. that suite references the representative Markdown roles `analysis` and
   `matching`,
3. the suite-definition shape is identical to the one used by other families.
