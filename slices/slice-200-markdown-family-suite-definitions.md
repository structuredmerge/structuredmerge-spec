# Slice 200: Markdown Family Suite Definitions

## Goal

Expose the Markdown family through the shared suite-descriptor mechanism.

## Shared Behavior

This slice defines one Markdown suite-definition contract:

1. a Markdown manifest MAY define one portable suite descriptor,
2. that descriptor references the representative Markdown roles `analysis` and
   `matching`,
3. the suite-definition shape is identical to the one used by other families.
