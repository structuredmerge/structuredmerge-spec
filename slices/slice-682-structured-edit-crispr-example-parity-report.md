# Slice 682: Structured Edit CRISPR Example Parity Report

## Goal

Standardize one portable report that states whether the active implementation
reproduces the selected legacy CRISPR examples.

## Shared Behavior

This slice defines one shared CRISPR example parity-report surface:

1. the report names the concrete example scenario being checked,
2. the report records whether the scenario is reproduced,
3. the report may carry implementation notes about remaining gaps.

## Notes

- This slice provides an explicit migration target from legacy examples into
  the new setup.
